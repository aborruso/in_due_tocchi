# Change: Add Telegram Template

## Why
Users need a template specifically designed for sharing content on Telegram groups, with Italian summaries that synthesize rather than translate the original content, using appropriate markdown formatting and emojis for better engagement.

## What Changes
- Add new "Telegram" template to templates.yaml
- Template creates original Italian titles and synthesized summaries optimized for Telegram sharing
- Uses markdown formatting (bold, code) and emojis for enhanced readability

## Impact
- Affected specs: template-system (new capability)
- Affected code: src/data/templates.yaml (add new template entry)
